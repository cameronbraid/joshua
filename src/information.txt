JavaScript array methods: https://ui.dev/c/advanced-javascript/javascript-array-methods-you-should-know


main JavaScript array methods:

    .map: creates new array with the result of calling provided function on every element in the array
    
    .filter: creates new array after filtering out elements that do not match the test provided by a function
    
    .every: used to determine if all elements pass the test provided by a function; if so, returns true, if not, returns false
    
    .includes: used if a particular value is found in an array; if so, returns true, if not, returns false
    
    .join: turns all elements of an array into a string and returns it separated with commas unless otherwise specified in parentheses
    
    .reverse: reverses the order of an array
    
    .some: used to determine if any element in the array passes the test provided by a function; if so, returns true, if not, returns false
